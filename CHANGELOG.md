# Changelog

## 0.3.0 - 2026-07-14

- Added `case-gallery` for deterministic Markdown/JSON public showcase comparisons.
- Added `visual-receipt` for deterministic SVG or HTML demo artifact receipts with hashes and routes.
- Added `reviewer-scorecard` for release evidence mapped to a maturity rubric.
- Added direct-holding/ETF-wrapper example fixtures in source examples and package data.
- Updated docs, skill guidance, tests, and demo artifacts for the public showcase layer.

## 0.2.0 - 2026-07-14

- Added `fixture-doctor` for deterministic Markdown/JSON fixture quality reports.
- Added optional `source_date` and `source_url` metadata to bundled examples and package data.
- Added checks for missing funds, non-1.0 weights, duplicate fixture rows, stale source metadata, and unheld constituent funds.
- Added tests and demo artifacts for the new fixture-doctor workflow.

## 0.1.0 - 2026-07-14

- Initial public-ready package scaffold.
- Added zero-runtime-dependency CLI with static look-through calculations.
- Added bundled examples, deterministic demo artifacts, tests, and public agent skill.
